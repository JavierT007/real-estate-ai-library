## Por qué lo guardé
La colección más grande: 4,343 workflows JSON, 365 integraciones, 15 categorías. Incluye buscador propio con filtros por categoría, complejidad y tipo de trigger.

## Cómo usarlo
Opción rápida (sin instalar nada):
- Buscar en https://zie619.github.io/n8n-workflows
- Descargar el JSON directo e importar en n8n

Opción local (búsqueda avanzada):
- docker run -p 8000:8000 zie619/n8n-workflows:latest
- Abrir http://localhost:8000
