## Por qué lo guardé
280+ plantillas n8n curadas por categoría: email (Gmail/Outlook), Telegram, WhatsApp, Slack, Discord, procesamiento de PDFs, agentes IA, RAG chatbots. Actualizado constantemente.

## Cómo usarlo
1. Navegar por carpeta de categoría en GitHub
2. Copiar el JSON del workflow
3. En n8n: Import from Clipboard
4. Ajustar credenciales antes de activar
5. Si se usa en un proyecto: copiarlo a workflows/ con ID WF-XXX
