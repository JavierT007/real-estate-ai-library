## Cómo usarlo
1. Inspiración: Awwwards, Godly, Land-book, Lapa Ninja
2. Componentes listos: Aceternity / Magic UI / React Bits (copiar + Framer Motion)
3. Fondos: BG Ibelick (gradientes animados en Tailwind)
4. 3D decorativo: Spline → export React → cargar con lazy() + Suspense

## Advertencias
- 3D y animaciones pesadas: siempre lazy() + Suspense, probar en móvil
- Spline plan free: marca de agua + hosting externo (verificar antes de producción)
- No mezclar GSAP y Framer Motion en el mismo componente
