# ArcatlasNext - Arquitectura

## ¿Qué es ArcatlasNext?

ArcatlasNext es una bóveda inteligente de conocimiento.

Su propósito es almacenar, organizar y reutilizar conocimiento para crear soluciones apoyadas por Inteligencia Artificial.

No está diseñado para una sola industria. Puede generar soluciones para cualquier tipo de negocio o necesidad.

---

## Filosofía

No desarrollamos software.

Diseñamos soluciones.

Cada solución genera nuevo conocimiento.

Ese conocimiento vuelve a ArcatlasNext para ser reutilizado.
