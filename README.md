# 🧠 ArcatlasNext — Bóveda Inteligente de Conocimiento

> Sistema de conocimiento reutilizable para diseñar soluciones con IA, automatización y arquitectura moderna — para cualquier industria.

## ¿Qué es?

ArcatlasNext almacena, organiza y reutiliza conocimiento técnico para que cada nuevo proyecto sea más rápido y mejor que el anterior.

No desarrollamos software. Diseñamos soluciones. Cada solución genera conocimiento que vuelve a la bóveda.

## Estructura

- **knowledge/** — Módulos técnicos (Supabase, n8n, PostgreSQL, UI/UX...)
- **solutions/** — Proyectos reales documentados (SOL-XXX) con lecciones aprendidas
- **resources/** — Recursos externos calificados (RES-XXX)
- **ai/** — Agentes, prompts, herramientas y memoria IA
- **workflows/** — Automatizaciones (Make, n8n, Pipedream)
- **prompts/** — Prompts reutilizables
- **docs/** — Documentación oficial y plantillas

## Sistema de IDs

| Prefijo | Tipo |
|---------|------|
| SOL-XXX | Solución/proyecto |
| RES-XXX | Recurso externo |
| DOC-XXX | Documento |
| WF-XXX  | Workflow |
| PRM-XXX | Prompt |
| AI-XXX  | Agente IA |

## Flujo de trabajo

1. Analizar el problema
2. Diseñar la solución
3. Documentarla
4. Desarrollarla
5. Probarla
6. Guardar el conocimiento (LESSONS.md)
