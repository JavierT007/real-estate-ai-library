# Arquitectura CRM Inmobiliario

## Stack principal

Frontend:
- Next.js
- React
- TypeScript
- Vercel

Backend:
- Supabase
- PostgreSQL
- Auth
- Storage

Automatización:
- n8n

Inteligencia Artificial:
- Claude
- OpenAI
- Gemini

## Flujo general

Usuario
 ↓
Sitio web Next.js
 ↓
Supabase
 ↓
n8n
 ↓
IA / APIs externas

## Objetivo

Crear un CRM inmobiliario con:
- Gestión de propiedades
- Gestión de clientes
- Seguimiento de leads
- Automatización de ventas
- Catálogo digital
- Agentes IA
