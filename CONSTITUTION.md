# Constitución de ArcatlasNext

1. Todo conocimiento debe documentarse.

2. Ninguna solución se desarrolla sin antes analizar el problema.

3. El código debe ser reutilizable.

4. Toda tecnología nueva debe documentarse.

5. Cada proyecto debe dejar nuevo conocimiento.

6. Nunca duplicar información.

7. La IA debe asistir, no reemplazar el criterio humano.
