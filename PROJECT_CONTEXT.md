# PROMPT DE CONTINUIDAD - ARCATLASNEXT

## Contexto

Estoy desarrollando un proyecto llamado **ArcatlasNext**.

Inicialmente nació como una biblioteca para un CRM inmobiliario, pero la visión cambió completamente.

ArcatlasNext YA NO es un CRM ni una plataforma inmobiliaria.

Su propósito es convertirse en una **Bóveda Inteligente de Conocimiento**, donde se almacena, organiza y reutiliza información para crear soluciones apoyadas por Inteligencia Artificial.

La idea es construir un sistema capaz de ayudar a desarrollar soluciones para cualquier tipo de empresa o necesidad.

Ejemplos:

- Tiendas
- Restaurantes
- Clínicas
- Inmobiliarias
- Fábricas
- Empresas de servicios
- Proyectos personales
- Automatizaciones
- Sistemas internos
- Aplicaciones móviles
- Sitios web
- APIs
- Dashboards

No está limitado a una industria.

---

# Filosofía

No desarrollamos software.

Diseñamos soluciones.

Cada solución genera nuevo conocimiento.

Ese conocimiento vuelve a ArcatlasNext para reutilizarse en proyectos futuros.

El objetivo es que cada nuevo proyecto sea más rápido y mejor que el anterior.

# Forma de trabajo

Queremos trabajar como si desarrolláramos un producto profesional.

Cada decisión importante debe documentarse.

Antes de escribir código:

1. Analizar el problema.

2. Diseñar la solución.

3. Documentarla.

4. Desarrollarla.

5. Probarla.

6. Guardar el conocimiento adquirido.

No queremos crear carpetas innecesarias.

La arquitectura debe ser escalable durante años.
